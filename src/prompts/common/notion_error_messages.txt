# Notion APIエラーメッセージ定義集

このファイルは、NotionツールAPIエラー発生時の統一的なエラーメッセージを定義します。
エージェントはこの形式に従い、ユーザーにわかりやすいエラー情報を提供してください。

## エラータイプと対応メッセージ

### トークン関連エラー

#### token_missing
```
❌ Notion API 接続エラー

Notion API トークンが設定されていません。
環境変数 NOTION_TOKEN を設定してください。

📋 **設定方法**
1. .env ファイルに以下を追加:
   NOTION_TOKEN=secret_xxxxxxxxxxxxxxxxxxxx
2. アプリケーションを再起動する
```

#### token_error
```
❌ Notion API 認証エラー

Notion API トークンが無効です。
有効なトークンを設定してください。

📋 **確認事項**
- トークンが正しいか
- トークンの有効期限が切れていないか
- 必要な権限が付与されているか
```

### パラメータ関連エラー

#### missing_parameter
```
❌ レシピ登録エラー

必須パラメータ（parent_id, title, properties）が不足しています。

📋 **考えられる原因**
- レシピ名（名前）が設定されていない
- 材料の情報が不足している
- 手順の情報が不足している
- データベースIDが正しく設定されていない

🔄 別のレシピまたはURLでもう一度お試しください。
```

#### validation_error
```
❌ レシピデータ検証エラー

レシピデータの形式に問題があります。

📋 **考えられる原因**
- 必須項目（名前、材料、手順）の形式が不正
- 数値項目に不正な値が設定されている
- データ構造に問題がある

🛠️ 別のレシピでもう一度お試しください。
```

### データベース関連エラー

#### database_id_missing
```
❌ データベース接続エラー

レシピデータベースIDが設定されていません。

📋 **確認事項**
- 正しいデータベースIDが設定されているか確認してください
- データベースが存在することを確認してください
- アクセス権限を確認してください
```

#### database_access_error
```
❌ データベースアクセスエラー

レシピデータベースにアクセスできません。

📋 **考えられる原因**
- データベースが存在しない
- 権限がない
- データベースIDが間違っている
```

### API関連エラー

#### api_error
```
❌ Notion API エラー

Notion API からエラーが返されました: [エラー詳細]

📋 **考えられる原因**
- API制限に達した
- サーバーエラーが発生した
- リクエスト形式に問題がある

⏱️ しばらく時間をおいてからもう一度お試しください。
```

#### unknown_error
```
❌ 不明なエラー

予期しないエラーが発生しました: [エラー詳細]

🔄 もう一度操作をお試しいただくか、別の方法でレシピを登録してください。
```

## エラーメッセージの使い分け

エラーの種類を正確に判断し、適切なメッセージテンプレートを選択してください。
不明なエラーの場合は、できるだけ詳細な情報をユーザーに提供してください。
すべてのエラーメッセージにおいて、ユーザーが取るべきアクションを明確に示してください。
